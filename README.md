# Watermark Tool

Simple web-based watermark editor built with HTML5 Canvas, Fabric.js, and TailwindCSS.

## Features

- Upload photo
- Upload custom logo watermark
- Built-in watermark logo presets
- Resize watermark
- Rotate watermark
- Opacity adjustment
- Analog joystick movement
- Mobile friendly
- Dark mode
- Export high-quality PNG

---

## Tech Stack

- HTML5
- [Fabric.js](http://fabricjs.com/) - Canvas manipulation.
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling.
- [Vanilla JavaScript] - Logic & Analog control.

---

## Usage

1. Upload photo
2. Upload or select watermark logo
3. Adjust size, opacity, and position
4. Download result
